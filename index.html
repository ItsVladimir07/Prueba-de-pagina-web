¿Que es el sonido Hi-Res?

El sonido Hi-Res o audio de alta definición es el medio de escuchar música de una forma cercana a como el artista lo concebió.

Este tipo de archivos generalmente se conocen como Lossless, o archivos sin pérdidas, donde estos conservan cada nota musical.

Existen diferentes tipos de archivos donde se pueden conservar estos archivos de audio, comenzando por uno llamado FLAC (O Free Lossless audio Code)

Este archivo es un archivo comprimido pero sin pérdidas de sonido, recortando su espacio pero haciendo que el sonido sea el mismo.

Existen otros tipos de formatos de audio?

Si, y se pueden explicar algunos:

Existe el WAV, que es un formato de audio base donde el audio no está comprimido, tiene toda la información y su tamaño es el doble del FLAC, Pero sigue siendo el mismo sonido.
